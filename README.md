# Pose-based Video Start Alignment on Mobile: Technical Note with Optional Audio-Assisted Refinement

## What this is / 概要
This repository publishes a **text-only technical note** describing an on-device method for aligning (synchronizing) two practice videos using pose-keypoint sequence matching, optionally assisted by audio-based candidate generation and/or audio verification (micro-refinement).

本リポジトリは、**Pose（骨格キーポイント）系列照合**により2本の練習動画の開始点（時間オフセット）を推定して同期再生する手法をまとめた**技術ノート（文章のみ）**です。必要に応じて **音声支援（粗候補生成）** と **音声検証／微調整（micro refinement）** を組み合わせる形態も含みます。

## Purpose / 目的
Released to help practitioners reproduce and improve the method, and to establish prior art for the described techniques.

実務者が再現・改善できる情報提供を目的に公開します。併せて、記載手法を**先行技術（prior art）として記録**する意図も含みます。
